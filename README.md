# YourEyes – 实时音频可视化与着色器沙盒

**YourEyes** 是一个基于 Qt6 / QML 和 OpenGL 的桌面应用，用于加载音频文件，实时分析频谱/波形，并通过可编程片段着色器（GLSL）生成动态视觉效果。它提供了灵活的采样器系统，可将音频特征（如频段能量）映射为着色器的 uniform 变量，实现真正的音频驱动视觉创作。

---

## 功能特性

- **音频加载与播放**  
  支持常见音频格式（MP3, WAV, FLAC, M4A, OGG 等），通过 Qt Multimedia 实现播放控制（播放/暂停、进度拖动）。

- **实时频谱 / 波形显示**  
  内置 STFT（短时傅里叶变换）分析，可同时显示原始波形、频谱和累积频谱（presum）。支持多通道音频。

- **可编程着色器系统**  
  - 内置默认频谱与波形着色器  
  - 支持从文件（`.frag`, `.fsh`）或源代码动态注册自定义片段着色器  
  - 着色器可接收 `m_time`（播放位置）、`m_resolution`、`ampRatio`、`zoom` 等内置 uniform  
  - 通过采样器系统，可注入自定义 uniform（如频段能量值）

- **采样器（Sampler）管理**  
  - 创建多个采样点，每个采样点可绑定到特定频段（范围选择）  
  - 实时计算该频段能量值，并作为 uniform 传递给当前激活的着色器  
  - 采样值可同时驱动多个视觉元素（如颜色、扭曲强度）

- **多窗口架构**  
  - 主窗口控制音频与采样器  
  - 每个频谱/着色器窗口独立运行，可同时打开多个视图，分别应用不同着色器

## 依赖与编译

### 依赖库
- **Qt 6**（必需模块：`qt6-base`, `qt6-declarative`, `qt6-multimedia`）
- **OpenGL** 开发库（系统自带）
- **dSTFT 库**（项目内置于 `Dependencies/STFT/`，需确保已编译）
- **FFTW 库** dSTFT的依赖项
- **libsndfile** dSTFT的依赖项

### 使用 Qt Creator 编译
1. 打开项目文件 `visual_studio.pro`（或 `CMakeLists.txt`，如果有）。
2. 配置编译套件（Desktop Qt 6.x MinGW/MSVC）
3. 确保 `Dependencies/STFT` 已正确构建并放入项目bin文件夹。
4. 点击“构建”即可。

## 使用方法

1. **启动应用**  
   运行编译后的可执行文件，主窗口显示简洁的控制界面。

2. **加载音频**  
   点击“选择文件”按钮，选择本地音频文件。加载完成后，音频会自动暂停，播放进度条可用。

3. **打开频谱窗口**  
   点击频谱图标，在弹出的列表中选择一个通道（如 `spectrum/channel-0`），即可打开频谱视图。  
   - 右键点击频谱区域可设置右边界（红色标记），左键设置左边界（黄色标记），用于采样范围。

4. **使用自定义着色器**  
   - 点击着色器图标（照片图标），打开着色器管理面板。  
   - 点击“+”按钮导入 `.frag` 文件，着色器将被注册到列表中。  
   - 点击列表中的着色器名称，即可打开一个新的着色器窗口，应用该着色器。

5. **采样器系统**  
   - 点击采样器图标（眼睛图标），打开采样器管理面板。  
   - 点击“+”按钮创建新的采样点，并命名。  
   - 在频谱窗口中用左右键划定采样范围（对应频段），该采样点将实时计算该频段能量。  
   - 在着色器窗口中，该值会作为 uniform（变量名与采样点名称相同）传递给着色器，可驱动视觉参数。

6. **控制参数**  
   - 每个视图窗口都配有“强度”和“缩放”滑块，可调整显示效果。  
   - 采样器面板提供最大能量调节和比率调节，以适配不同动态范围。

## 扩展开发

- **新增着色器**：将 `.frag` 文件置于任意目录，通过主界面“导入”即可使用。着色器必须包含 `main()` 函数，并可使用以下 uniform（按需）：
  ```glsl
  uniform float m_time;           // 播放位置（毫秒）
  uniform vec2  m_resolution;     // 窗口分辨率（像素）
  uniform float zoom;            // 缩放系数
  uniform float ampRatio;        // 幅度倍率
  uniform sampler2D spct;        // 频谱纹理（仅频谱/波形模式可用）
  // 自定义采样器变量（名称与采样点一致，类型为 float）
  uniform float sampler_0;
  ```

- **添加新采样点**：在采样器面板创建，然后在频谱窗口中用鼠标划取频段范围，该采样点即与频谱绑定。

- **修改内置着色器**：可在 `shadermanager.cpp` 中修改默认片段着色器源码。
